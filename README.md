# Simple Networking Chat Project

This project demonstrates a basic client-server chat application using Python sockets. 
It covers fundamental networking concepts such as:

- TCP connections
- Client-server communication
- IP and Port usage

## How to Run

1. Run `server.py` first:
2.  Run `client.py` in another terminal or machine:
3.  3. Start chatting between client and server!
